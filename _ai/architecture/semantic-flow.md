# Semantic Flow — модель движения смыслов в author-mind

Semantic Flow (SF) — это динамическая система циркуляции смыслов,  
которая превращает статичную структуру author-mind в **живой поток**.

SE (Semantic Engine) — вычисляет смысловые изменения.  
SF — определяет, как смыслы течёт через всю систему.

---

# 1. Что такое "течение смыслов"

Под **течением** понимается:

- перемещение смыслов между сущностями  
- их трансформация под влиянием контекста  
- формирование новых линий, тем, миров  
- миграция интуиций в идеи, а идей — в линии  
- обратный поток: линии → идеи → миры → экспорты  

Смысл не статичен. Он **движется**, изменяя всё поле.

---

# 2. Семь уровней смыслового потока

Потому что любая мысль проходит путь зрелости.

intuition → idea → theme → line → story → world → export

### 2.1 intuition  
микро-озарение, импульс

### 2.2 idea  
оформленная концепция

### 2.3 theme  
категория, вокруг которой собираются идеи

### 2.4 line  
зрелый смысловой поток, непрерывная мысль

### 2.5 story  
эпизод, повествовательная упаковка линии

### 2.6 world  
контейнер: сеттинг, образ, мифологема

### 2.7 export  
передача в другие репозитории в форме:

- текста  
- архитектуры  
- документации  
- художественного материала  

---

# 3. Внутренние направления потока

## 3.1 Восходящий поток  
из хаоса → в структуру  
интуиция → идея → тема → линия → история → мир → экспорт

## 3.2 Нисходящий поток  
структуры → в новые интуиции  
миры → истории → линии → идеи → интуиции  

Так работает реальная мысль.

## 3.3 Циркуляция  
когда не появляется ничего нового,  
а существующие смыслы усиливают друг друга.

---

# 4. Правила движения смыслов

### Правило 1  
Смысл движется туда, где есть незаполненное пространство.

### Правило 2  
Сильный смысл стремится соединиться с другим сильным.

### Правило 3  
Если связи слишком плотны — смысл ветвится.

### Правило 4  
Если связей нет — смысл умирает или превращается в "тёмную область".

---

# 5. Архитектура Semantic Flow

SF состоит из трёх модулей:

## 5.1 Flow Tracker  
отслеживает перемещения смыслов между файлами.

## 5.2 Flow Predictor  
предсказывает “куда пойдёт смысл дальше”.

## 5.3 Flow Synthesizer  
предлагает:

- новые связи
- новые идеи
- новое развитие линий
- расширение миров
- точки экспорта в проекты

---

# 6. Роль автора

Система предлагает → Автор направляет  
Это совместная мыслительная динамика.
